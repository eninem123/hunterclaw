## 一、主板策略（价值投资）

### 1.1 选股标准

| 指标 | 阈值 | 说明 |
|------|------|------|
| PE(TTM) | ≤15 | 低估值，防御性强 |
| 股息率 | ≥3% | 高分红，现金回报 |
| ROE | ≥10% | 盈利能力稳定 |
| 营收增速 | ≥5% | 成长性保障 |
| 资产负债率 | ≤60% | 财务杠杆可控 |

### 1.2 买入信号

```
买入条件：
✓ PE ≤ 15 且 股息率 ≥ 3%
✓ ROE ≥ 10%
✓ 近30日主力净流入 > 0
✓ 股价处于MA20之上
✓ 市场温度 ≥ 40℃
```

### 1.3 卖出信号

```
卖出条件：
✗ PE > 25（估值偏高）
✗ 股息率 < 1.5%（分红下降）
✗ 股价跌破MA60
✗ 主力连续净流出 > 5日
✗ 市场温度 < 25℃
```

### 1.4 典型标的

- 长江电力(600900)：水电龙头，高分红典范
- 中国神华(601088)：煤电一体化，高股息
- 大秦铁路(601006)：稳定现金流，高分红
- 工商银行(601398)：低估值银行，防御性强

## 四、ETF策略（轮动策略）

### 4.1 核心逻辑

ETF不存在个股黑天鹅，但存在**折溢价套利机会**和**行业轮动机会**。

### 4.2 折溢价套利

```
判断标准：
- 溢价率 > 0.5%：卖出ETF，申购套利（机构行为）
- 折价率 > 0.5%：买入ETF，赎回套利（机构行为）

数据来源：
- 东财ETF数据中心
- 东财LOF基金数据中心
```

### 4.3 行业轮动信号

```
买入信号：
✓ 相关行业指数涨幅 > 3%
✓ 量比 > 1.5x
✓ 板块资金净流入 > 10亿
✓ 成分股涨停家数 ≥ 3家

卖出信号：
✗ 行业指数跌破MA20
✗ 主力资金净流出 > 20亿
✗ 情绪退潮（跌停家数 > 5家）
```

### 4.4 典型标的

- 稀土ETF(159715)：政策+供需双驱动
- 芯片ETF(159995)：国产替代主线
- 新能源ETF(516160)：碳中和长期逻辑
- 医疗ETF(512010)：估值修复机会

## 五、基金策略（选基逻辑）

### 5.1 选基标准

| 维度 | 指标 | 阈值 |
|------|------|------|
| 收益 | 近1年超额收益 | ≥5% |
| 风控 | 最大回撤 | ≤-20% |
| 稳定性 | 收益标准差 | ≤15% |
| 经理 | 从业年限 | ≥5年 |
| 规模 | 基金规模 | 10-100亿 |

### 5.2 经理风格分类

```
价值型：
- 重仓低PE、高股息股票
- 换手率 < 100%
- 典型：曹名长、丘栋荣

成长型：
- 重仓高增速、高弹性股票
- 换手率 > 200%
- 典型：冯明远、李晓星

均衡型：
- 价值+成长配置
- 换手率 100-200%
- 典型：周蔚文、袁芳

行业型：
- 单一行业集中
- 跟随行业周期
- 典型：赵诣（新能源）、葛兰（医药）
```

### 5.3 买入基金信号

```
买入条件：
✓ 经理从业 ≥ 5年
✓ 近1年超额收益 ≥ 5%
✓ 最大回撤 ≤ -20%
✓ 当前处于行业低位（板块PE历史分位 ≤ 30%）
✓ 基金经理自购（跟投信号）
```

### 5.4 卖出基金信号

```
卖出条件：
✗ 连续2年跑输基准 > 10%
✗ 经理离职（换人风险）
✗ 规模急剧缩水 < 1亿（清盘风险）
✗ 持仓行业估值泡沫化
✗ 最大回撤 > -30%（风控失控）
```
