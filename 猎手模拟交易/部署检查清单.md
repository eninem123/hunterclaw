## 🚀 部署步骤

### 1. 停止旧服务（如果运行中）
```bash
systemctl stop mcp-data-server
# 或
pkill -f "api_server"
```

### 2. 更新代码
```bash
cd /root/.openclaw/workspace/猎手模拟交易
git pull  # 如果使用git
# 或手动复制新文件
```

### 3. 安装依赖
```bash
cd /root/.openclaw/workspace/猎手模拟交易
source venv/bin/activate  # 或创建新虚拟环境
pip install -r requirements.txt

# MCP服务器依赖
cd mcp-data-server
source venv/mcp-data-server/bin/activate
pip install -r requirements.txt
```

### 4. 配置调整
```bash
# 修改API Token
vi mcp-data-server/src/api_server_v2.py

# 修改配置（可选）
vi config.json
```

### 5. 启动服务
```bash
# 启动MCP服务器
cd /root/.openclaw/workspace/猎手模拟交易/mcp-data-server
source venv/mcp-data-server/bin/activate
nohup python -m uvicorn src.api_server_v2:app --host 0.0.0.0 --port 8766 > /var/log/mcp-data-server.log 2>&1 &

# 或使用systemd
systemctl start mcp-data-server
systemctl enable mcp-data-server
```

### 6. 更新Cron任务
```bash
crontab -e
# 修改为：
40 15 * * 1-5 cd /root/.openclaw/workspace/猎手模拟交易 && /root/.openclaw/workspace/猎手模拟交易/venv/bin/python hunter_closed_loop_v2.py >> /var/log/hunter_loop.log 2>&1
```
