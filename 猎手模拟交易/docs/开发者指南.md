## 系统概述

猎手系统是一个基于OpenClaw框架的A股智能交易系统，采用多Agent协作架构，实现自动选股、信号处理、交易执行和复盘分析。

### 核心特性

- **自动选股**：基于技术指标和市场情绪的智能选股
- **信号处理**：实时监控持仓，自动触发止损止盈
- **策略学习**：从交易历史中提取经验，不断优化策略
- **风险控制**：多层风险防护机制
- **数据服务**：统一的数据接口，支持多种数据源

### 技术栈

| 组件 | 技术 | 版本 |
|------|------|------|
| 主框架 | OpenClaw | 2026.4.x |
| 编程语言 | Python | 3.8+ |
| Web框架 | FastAPI | 0.100+ |
| 数据处理 | Pandas, NumPy | 最新版 |
| 异步框架 | asyncio, aiohttp | - |
| 缓存 | 内存缓存（支持Redis扩展） | - |

## 模块说明

### 1. 闭环系统 (hunter_closed_loop_v2.py)

#### 功能
- 读取市场数据和持仓信息
- 分析市场情绪和趋势
- 更新策略（写入教训/经验）
- 扫描明日候选股票
- 持仓复盘
- 生成日终总结

#### 主要函数

```python
def run_closed_loop(config: HunterConfig = None) -> Dict:
    """运行完整闭环"""
    
def step1_read_data(config: HunterConfig) -> Dict:
    """Step 1: 读取所有数据"""
    
def step2_analyze(data: Dict, config: HunterConfig) -> Dict:
    """Step 2: 分析今日市场"""
    
def step3_update_strategy(data: Dict, analysis: Dict, config: HunterConfig) -> List[str]:
    """Step 3: 更新策略，写入知识库"""
    
def step4_scan_for_tomorrow(max_count: int = 3, config: HunterConfig = None) -> List[Dict]:
    """Step 4: 扫描明日候选"""
    
def step5_review_positions(data: Dict, config: HunterConfig) -> Dict:
    """Step 5: 持仓复盘"""
    
def step6_summary(reviews: List[Dict], learnings: List[str], 
                 analysis: Dict, tomorrow_candidates: List[Dict]) -> Tuple[str, str]:
    """Step 6: 生成今日总结"""
```

#### 使用示例

```python
from hunter_closed_loop_v2 import run_closed_loop, HunterConfig

# 使用默认配置

# 使用自定义配置

    timeout=10,
    max_retries=5,
    max_candidates=5
)
result = run_closed_loop(config)
```

### 2. 数据分析模块 (data_analyzer.py)

#### 功能
- 技术指标计算（MA、RSI、MACD、布林带、KDJ）

#### 主要类

```python
class DataAnalyzer:
    """数据分析器"""
    
    def calculate_ma(self, prices: np.ndarray, period: int) -> Optional[float]:
        """计算移动平均线"""
        
    def calculate_rsi(self, prices: np.ndarray, period: int = 14) -> Optional[float]:
        """计算RSI指标"""
        
    def calculate_macd(self, prices: np.ndarray) -> Tuple[Optional[float], Optional[float], Optional[float]]:
        """计算MACD指标"""
        
    def analyze_stock(self, code: str, name: str, kline_data: List[Dict], 
                     current_price: float = None, chg_pct: float = 0.0, 
                     volume_ratio: float = None) -> AnalysisResult:
        """分析单只股票"""
        
    def batch_analyze(self, stocks_data: List[Dict]) -> List[AnalysisResult]:
        """批量分析股票"""
```

#### 使用示例

```python
from data_analyzer import DataAnalyzer

# 创建分析器

# 分析单只股票
result = analyzer.analyze_stock(
    code='000001',
    name='平安银行',
    kline_data=kline_data,
    current_price=10.8,
    chg_pct=2.5,
    volume_ratio=1.5

print(f"趋势: {result.trend.value}")
print(f"信号: {result.signal.value}")
```

### 3. MCP数据服务器 (api_server_v2.py)

#### 功能
- 提供统一的HTTP API接口
- 支持实时行情、K线数据、市场概览
- 内置缓存和限流机制
- 完善的错误处理

#### API端点

| 端点 | 方法 | 描述 |
|------|------|------|
| `/api/v2/health` | GET | 健康检查 |
| `/api/v2/quote/{symbol}` | GET | 获取实时行情 |
| `/api/v2/kline/{symbol}` | GET | 获取K线数据 |
| `/api/v2/market/overview` | GET | 获取市场概览 |
| `/api/v2/cache/stats` | GET | 获取缓存统计 |
| `/api/v2/cache/clear` | POST | 清空缓存 |

#### 使用示例

```bash
# 健康检查
curl -H "X-API-Token: mcp-data-server-token" \
  http://localhost:8766/api/v2/health

# 获取实时行情
curl -H "X-API-Token: mcp-data-server-token" \
  http://localhost:8766/api/v2/quote/000001

# 获取K线数据
curl -H "X-API-Token: mcp-data-server-token" \
  "http://localhost:8766/api/v2/kline/000001?start_date=20240101&end_date=20241231"

# 获取市场概览
curl -H "X-API-Token: mcp-data-server-token" \
  http://localhost:8766/api/v2/market/overview
```

### Q1: 如何添加新的技术指标？

A: 在 `data_analyzer.py` 中添加计算函数：

```python
def calculate_atr(self, highs: np.ndarray, lows: np.ndarray, 
                 closes: np.ndarray, period: int = 14) -> Optional[float]:
    """计算ATR指标"""
    if len(closes) < period + 1:
        return None
    
    tr_list = []
    for i in range(1, len(closes)):
        tr = max(
            highs[i] - lows[i],
            abs(highs[i] - closes[i-1]),
            abs(lows[i] - closes[i-1])
        )
        tr_list.append(tr)
    
    return float(np.mean(tr_list[-period:]))
```
