## 核心发现

### 1. 资金流向量化标准

| 订单分类 | 金额标准 | 投资者类型 |
|---------|---------|-----------|
| 小单 | <6万元 | 散户 |
| 中单 | 6-30万元 | 中小投资者 |
| 大单 | 30-90万元 | 主力试探 |
| 特大单 | ≥90万元 | 主力核心操作 |

**核心公式**：
```
主力净流入额 = (特大单买入金额 + 大单买入金额) - (特大单卖出金额 + 大单卖出金额)
主力净流入占比 = (主力净流入额 / 当日总成交额) × 100%
```

### 2. 四阶段量化信号

| 阶段 | 主力净流入占比 | 特征 |
|------|--------------|------|
| 吸筹期 | 10%-20% | 大单持续净流入，缩量或温和放量 |
| 洗盘期 | 5%-15% | 股价下跌但大单净流入≥0 |
| 拉升期 | ≥20% | 大单持续大幅净流入，量价齐升 |
| 出货期 | ≤-10% | 大单持续净流出，股价涨但资金流出 |

### 3. 主力欺骗手法识别

**手法一：大单买入+小单卖出**
- 表现：主力资金"净流入"
- 本质：实际正在出货
- 识别：需结合内外盘验证

**手法二：小单买入+大单卖出**
- 表现：主力资金"净流出"
- 本质：实际正在吸筹
- 识别：股价下跌但大单净流入≥0

**交叉验证方法**：
- 外盘 > 内盘 + 股价不涨 → 主力出货
- 外盘 < 内盘 + 股价不跌 → 主力吸筹

### 4. 北向资金择时指标

**中金公司研究结论**：
- 北向净流入 > 500亿元 → 看空信号（市场过热）
- 北向净流出 < -500亿元 → 看多信号（市场过度悲观）
- 短周期看：连续3天以上同向流动更有趋势意义
- 个股参考：单日北向流入占流通盘≥0.5%算大幅
