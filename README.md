## 🎯 项目简介

HunterClaw 是一套**智能A股交易系统**，通过三层AI Agent协作：
- **Claw（执行者）**：交易执行、代码操作、微信推送
- **Hermes（顾问）**：策略分析、记忆整理、复盘优化
- **GBrain（记忆）**：知识管理、语义搜索、经验沉淀

核心特点：**市场温度模型** + **自动交易执行** + **严格止损机制**

## 🧠 核心模块

### 1. auto_trade.py - 自动交易引擎

市场温度模型判断系统状态，自动执行买卖：

```python
# 市场温度阈值
TEMPICY_THRESHOLD = 30   # 冰点（缩量）
WARM_THRESHOLD = 50      # 暖意（可建仓）
STOP_LOSS_PCT = 5        # 止损-5%（红线）
TAKE_PROFIT_PCT = 10     # 止盈+10%
```

**触发条件**：
| 操作 | 条件 |
|------|------|
| 买入 | 市场温度≥50℃ + 放量≥5% + 无熔断 |
| 卖出 | 止损-5% / 止盈+10% / 风控熔断 |

### 2. portfolio_tracker.py - 持仓管理

- 持仓买入/卖出记录
- 止损止盈线计算
- 浮盈浮亏实时更新

### 3. 市场温度模型

基于成交量变化判断市场情绪：

| 温度 | 成交量变化 | 操作 |
|------|-----------|------|
| <30℃ 冰点 | 缩量>20% | 观望 |
| 30-50℃ | 缩量5-20% | 等待 |
| 50-65℃ 暖意 | 持平~放量15% | 可建仓 |
| 65-80℃ | 放量15-30% | 积极做多 |
| >80℃ 强势 | 放量>30% | 满仓操作 |

## 🚀 快速开始

### 环境要求

- Python 3.12+
- 腾讯行情API（用于获取实时股价）

### 安装依赖

```bash
pip install akshare
```

### 运行自动交易检测

```bash
cd hunterclaw
python3 src/auto_trade.py
```

### 生成持仓报告

```bash
python3 src/report_generator.py 持仓
```

## 🔄 定时任务

| 时间 | 任务 | 说明 |
|------|------|------|
