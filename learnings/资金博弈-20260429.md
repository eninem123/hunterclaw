# 资金博弈深度学习记录

## 核心发现

### 1. 资金流向量化标准

| 订单分类 | 金额标准 | 投资者类型 |
|---------|---------|-----------|
| 小单 | <6万元 | 散户 |
| 中单 | 6-30万元 | 中小投资者 |
| 大单 | 30-90万元 | 主力试探 |
| 特大单 | ≥90万元 | 主力核心操作 |

**核心公式**：
- 主力净流入额 = (特大单买入金额 + 大单买入金额) - (特大单卖出金额 + 大单卖出金额)
- 主力净流入占比 = (主力净流入额 / 当日总成交额) × 100%

### 2. 四阶段量化信号

| 阶段 | 主力净流入占比 | 特征 |
|------|--------------|------|
| 吸筹期 | 10%-20% | 大单持续净流入，缩量或温和放量 |
| 洗盘期 | 5%-15% | 股价下跌但大单净流入≥0 |
| 拉升期 | ≥20% | 大单持续大幅净流入，量价齐升 |
| 出货期 | ≤-10% | 大单持续净流出，股价涨但资金流出 |

### 3. 主力欺骗手法识别

**手法一：大单买入+小单卖出**
- 表现：主力资金净流入
- 本质：实际正在出货
- 识别：需结合内外盘验证

**手法二：小单买入+大单卖出**
- 表现：主力资金净流出
- 本质：实际正在吸筹
- 识别：股价下跌但大单净流入≥0

### 4. 北向资金择时指标

- 北向净流入 > 500亿元 → 看空信号（市场过热）
- 北向净流出 < -500亿元 → 看多信号（市场过度悲观）
- 短周期：连续3天以上同向流动更有趋势意义
- 个股参考：单日北向流入占流通盘≥0.5%算大幅

## 策略更新建议

### 新增量化标准
1. 主力净流入占比 > 20% + 股价上涨 → 确认拉升，可追涨
2. 主力净流入占比 < -10% → 确认出货，立即止损
3. 北向连续3日净流入 → 趋势确认信号

### 新增风控规则
- 股价涨 + 主力净流出 → 诱多预警，禁止买入
- 股价跌 + 主力净流入 → 低位吸筹，可小仓试探
