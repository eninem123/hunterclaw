# 猎手交易策略说明

> 版本：v3.69 | 更新：2026-04-29 | 核心策略体系

---

## 一、核心规则体系（R1-R9, R49）

### R1: 整数关口预警机制

```
触发条件：指数接近整数关口（±0.5%范围内）
- 增加风险权重，不盲目追高
- 观察突破质量：是否缩量、是否站稳30分钟+

突破后快速回落信号（必须减仓）：
1. 突破后30分钟内跌回关口下方
2. 同时下跌家数 > 3000家
3. 主力资金净流出 > 300亿
```

### R2: 市场赚钱效应仓位系数

```
| 上涨:下跌比例 | 仓位上限 | 说明 |
|---------------|----------|------|
| > 2:1 | 6-7成 | 强势市场，可重仓 |
| 1:1 ~ 2:1 | 4-5成 | 中性市场，半仓操作 |
| 1:1.5 ~ 1:2 | 2-3成 | 偏弱市场，轻仓 |
| < 1:2 | 1-2成 | 弱势市场，空仓/极轻仓 |

触发：每日14:00检查涨跌家数比
```

### R3: 主力诱多识别（立即减仓信号）

```
任一出现即减仓至3成以下：
1. 指数微涨（<0.5%）但上涨家数 < 1500家
2. 主力净流出 > 500亿 且 指数跌幅 < 0.5%
3. 成交额放大（>2.5万亿）但指数不涨
4. 防御板块（银行/电力/白酒）逆势走强
5. 散户小单净流入 > 300亿（主力出、散户接）
```

### R4: 事件催化风险强化

```
前置检查：
□ 一季报预披露时间（4月底截止）
  - 业绩下滑 > 30% → ❌ 禁止买入
  - 业绩不确定（未披露）→ ⚠️ 降仓至1成

事件驱动交易铁律：
❌ 禁止在事件日前3天内买入（利好提前透支风险）
✅ 事件日前5-7天买入，事件日前1-2天必须止盈
✅ 事件当天开盘即清仓，不抱幻想

事件前最后2天规则：
- 事件前2天：持仓必须设止盈位（成本价+3%）
- 事件当天：不新买入，已有仓位考虑止盈
- 事件后次日：无论盈亏，优先出局

事件股止损规则：
- 事件驱动股止损位：成本价-5%（比普通股更严格）
- 低开低走形态：当日减仓50%，不必等止损位
```

### R5: 五一前特殊规则

```
4/25前最后5个交易日：
- 总仓位降至3成以下
- 避免节前流动性收缩风险
- 不新开仓，等五一后局势明朗
```

### R6: 情绪极值仓位锁定

```
情绪极值是A股短期顶底的精准先行指标：

| 恐贪指数 | 仓位上限 | 操作 |
|----------|----------|------|
| ≥90分 | 0成 | 空仓，等待 |
| 85-90分 | 2成 | 极轻仓，分批止盈 |
| 70-85分 | 5成 | 逐步止盈，控制风险 |
| 30-70分 | 5-6成 | 正常操作 |
| 20-30分 | 4成 | 试探性布局 |
| ≤20分 | 等待 | 等回升至20%以上右侧入场 |

触发条件：每日收盘前检查恐贪指数
配合R2（赚钱效应仓位系数）使用，取两者较低值
```

### R7: 情绪周期五阶段仓位管理

```
情绪周期是A股短线操作的核心参考：

| 阶段 | 特征 | 涨停家数 | 仓位上限 | 操作策略 |
|------|------|----------|----------|----------|
| 启动期 | 冰点后低位连板出现，跌停减少 | <50家 | 1-2成 | 轻仓试错首板 |
| 发酵期 | 龙头确立(3板+)，板块梯队完整 | 50-100家 | 3-5成 | 加仓核心龙头 |
| 高潮期 | 百股涨停，龙头缩量加速 | >100家 | 逐步止盈 | 持有龙头，卖后排 |
| 退潮期 | 龙头断板，跌停增加，炸板率升 | 30-50家 | 1-2成 | 清仓观望 |
| 冰点期 | 跌停>50家，市场极度悲观 | <30家 | 空仓 | 等待冰点信号 |

核心口诀：启动轻仓试错，高潮分批止盈，退潮空仓休息
```

### R8: 龙头股筛选标准

```
连板龙头识别条件：
1. 连板天数：≥2天（最好3板+）
2. 板块联动：同板块涨停≥3家
3. 筹码集中：缩量加速代表主力锁仓
4. 市场地位：情绪周期处于发酵期或高潮期

龙头股操作原则：
- 只做最强的龙头，不做跟风
- 缩量一字板次日高开可打板
- 放巨量炸板当日必须减仓
- 龙头首阴可博弈反包
```

### R9: 非对称买卖点

```
核心逻辑：在情绪极端时逆向操作

买点（左侧）：
- 恐贪指数≤20分，市场冰点
- 连续下跌后出现缩量十字星
- 板块龙头回调至10日均线附近

卖点（右侧）：
- 恐贪指数≥85分，市场极度贪婪
- 个股放巨量突破前高
- 板块出现批量减持公告

止损铁律：
- 普通股：成本价-7%
- 事件驱动股：成本价-5%
- 龙头股：跌破5日线减半，跌破10日线清仓
```

## 二、技术分析规则

### R35: 低位启动信号

```
触发条件（需同时满足）：
1. 回撤幅度 > 25%（距180日高点）
2. 量比 > 1.5x（相对近期均量）
3. 涨幅 > 7%（当日涨幅）

信号意义：
- 主力低位吸筹后启动
- 可能形成反弹或反转
- 需配合板块联动判断

分析参考：
- 首次信号：轻仓试探（1-2成）
- 二次确认：加仓至3-5成
- 止损位：信号日最低价
```

### R38: 连板龙头信号

```
触发条件（需同时满足）：
1. 连板天数 ≥ 2天
2. 同板块涨停家数 ≥ 3家
3. 情绪周期处于发酵期或高潮期

信号意义：
- 短线最强赚钱效应
- 主力强势拉升
- 情绪周期上升阶段

分析参考：
- 持有策略：持有不动，等放量炸板
- 买入策略：情绪高潮期不做追高
- 卖出策略：缩量加速次日高开低走即卖
```

## 三、风控规则

### 仓位计算公式

```
仓位参考 = MIN(R9-v2温度仓位, R2仓位, R6仓位, R7仓位, R4事件仓位)

R9-v2温度仓位矩阵（2026-05-16升级）：
| 市场温度 | 仓位上限 |
|---------|---------|
| ≥80℃（过热） | 0-10% |
| 70-80℃（高温） | 20% |
| 50-70℃（正常） | 50% |
| 35-50℃（偏弱） | 30% |
| 25-35℃（冰点） | 10% 🚫禁止新开仓|
| <25℃（恐慌） | 0% 🚨强制空仓 |

示例：
- 温度35℃冰点 → R9-v2仓位=10%
- 赚钱效应1:1 → R2仓位=4成
- 恐贪指数82分 → R6仓位=5成
- 情绪退潮期 → R7仓位=1-2成
- 无重大事件 → R4仓位=正常

取最小值作为最终仓位上限
```

### 浮亏分级响应规则（R20-v4.0）

```
| 浮亏幅度 | 等级 | 操作 |
|----------|------|------|
| 0~-2% | 🟢 绿色 | 正常持有，设置预警 |
| -2%~-4% | 🟡 黄色 | 检查大盘/板块，准备减仓 |
| -4%~止损 | 🟠 橙色 | 设价格提醒，准备执行止损 |
| 触止损 | 🔴 红色 | 无条件执行，不犹豫 |

改进：在-2%时主动评估，不等硬止损触发
```

### 批量建仓冷却规则（R21-v4.0）

```
单日限：≤2只（原3只），单只≤20%（原30%）
板块限：同板块≤30%，相关板块≤40%
冷却期：买入1只后等待≥30分钟
违规惩罚：超限后系统锁定买入30分钟
```

### 止损位计算（R07-v2升级）

```
基础止损位 = 买入价 × (1 - 基准止损%) × 恐慌因子(F)

基准止损%（温度×持仓天数矩阵）：
| 温度 | 1-3天 | 4-7天 | 8天+ |
|------|-------|-------|------|
| ≥70℃ | 5.0% | 7.0% | 10.0% |
| 50-70℃ | 4.0% | 6.0% | 8.5% |
| 35-50℃ | 3.5% | 5.0% | 7.0% |
| <35℃ | 3.0% | 4.0% | 5.5% |

恐慌因子F（2026-05-16新增）：
  恐慌指数≥25 → F=1.0（正常）
  恐慌指数15-25 → F=0.85（收紧15%）
  恐慌指数<15 → F=0.70（收紧30%，仅减仓）

ATR调整：
  ATR(14) > 均价×4% → 止损放宽1.2倍

示例：冰点期(30℃)持仓3天，恐慌指数14 → F=0.70
  止损=买入价×(1-3.0%×0.70)=买入价×0.979（-2.1%）
  理由：极端环境下缩小持仓容忍度，更快离场

特例：
  普通股：买入价 × (1-基准止损%×F)
  事件驱动股：买入价 × (1-基准止损%×F×0.85)
  龙头股：5日均线（动态，取两者较严格）

止损执行：
1. 触及止损位立即执行，不犹豫
2. R20: -2%预警，-3%主动减仓，-4.5%准备止损
3. 连续3日下跌无论是否到止损位都需评估
```
