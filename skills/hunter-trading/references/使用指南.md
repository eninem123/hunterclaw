## 二、功能详解

### 2.1 股票查询（¥0.02）

**触发方式**：
- "查一下XXX"
- "XXX的行情"
- "XXX现在多少钱"

**输出内容**：
- 股票名称、代码
- 当前价格、涨跌幅
- 成交量、成交额
- 行业分类、总市值

**示例**：
```
输入：查一下宁德时代
输出：
宁德时代(300750)
当前价：185.60元 | 涨跌幅：+2.35%
成交量：28.5万手 | 成交额：52.8亿
行业：新能源汽车
总市值：约8650亿

🔓 查看详细分析 ¥0.05
```

### 2.2 标准分析（¥0.05）

**触发方式**：
- "分析XXX"
- "帮我看看XXX"
- "XXX的技术面怎么样"

**输出内容**：
- 行情概览（价格、涨跌幅、止损价）
- 技术分析（信号、形态、关键价位）
- 情绪判断（恐贪指数、情绪阶段、仓位参考）
- 风险评估（主力资金、诱多风险）
- 分析参考（买入/持有/卖出）

**示例**：
```
输入：分析航天电子
输出：
# 猎手分析报告 - 航天电子

## 一、行情概览
| 项目 | 数值 |
|-----|------|
| 当前价 | 23.74元 |
| 涨跌幅 | -5.12% |
| 止损价 | 23.75元 |
| 状态 | ⚠️ 需关注 |

## 二、技术分析
- 信号：🔴 看跌
- 回撤幅度：28.5%（距180日高点）
- 形态：下降趋势

## 三、情绪判断
- 恐贪指数：79.4分
- 情绪阶段：高潮期→退潮期
- 仓位参考：≤2成

## 四、风险评估
- 主力资金：连续净流出
- 诱多风险：需警惕

## 五、分析参考
🎯 参考：谨慎观望

⚠️ 风险提示：本报告仅供模拟交易参考，不构成投资参考。
```

### 2.3 持仓诊断（¥0.05）

**触发方式**：
- "帮我看看我的持仓"
- "诊断一下持仓"
- "我的股票怎么样"

**输入格式**：
```
持仓：股票1代码/成本, 股票2代码/成本, ...
```

**输出内容**：
- 各持仓风险评估
- 综合仓位参考
- 调仓参考
- 止损提醒

**示例**：
```
输入：诊断我的持仓：比亚迪/180, 宁德时代/200, 隆基绿能/25
输出：
# 持仓诊断报告

## 一、持仓概览
| 股票 | 成本价 | 现价 | 盈亏 | 风险等级 |
|-----|-------|------|------|----------|
| 比亚迪 | 180元 | 172元 | -4.4% | 中 |
| 宁德时代 | 200元 | 195元 | -2.5% | 低 |
| 隆基绿能 | 25元 | 23元 | -8.0% | 高 |

## 二、综合评估
- 总仓位：5成
- 整体风险：中等偏高
- 赚钱效应：偏弱（1:1.2）

## 三、调仓参考
1. 隆基绿能触及止损位，减仓参考50%
2. 比亚迪可持有但不加仓
3. 宁德时代相对稳健，可持有

## 四、仓位参考
当前仓位参考：3-4成
总持仓参考降至3成左右

⚠️ 风险提示：本报告仅供模拟交易参考，不构成投资参考。
```

### 2.4 止损计算（¥0.05）

**触发方式**：
- "止损多少"
- "帮我算下止损"
- "XXX的止损位在哪"

**输入格式**：
```
止损：股票代码/买入成本
```

**输出内容**：
- 普通止损位（-7%）
- 事件驱动止损位（-5%）
- 关键支撑位
- 止盈位计算

### 2.5 仓位查询（¥0.02）

**触发方式**：
- "现在仓位多少合适"
- "今天怎么看"
- "市场情绪怎么样"

**输出内容**：
- 赚钱效应评估
- 恐贪指数
- 情绪周期阶段
- 仓位参考

## 四、常见问题

### Q1: 支持哪些股票？
A: 支持沪深两市所有A股，包括主板、创业板、科创板。

### Q2: 数据更新频率？
A: 行情数据实时更新，情绪数据每日收盘更新。

A: 本系统基于历史数据和规则模型进行分析，仅供参考，不构成投资参考。
