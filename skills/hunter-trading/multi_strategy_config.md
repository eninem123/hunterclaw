## 各板块筛选规则

### 主板（价值投资）
- PE ≤ 15
- 股息率 ≥ 3%
- ROE ≥ 10%
- 主力净流入 > 0

### 创业板（趋势跟踪）
- 涨幅 5-9.8%（排除涨停）
- 量比 > 1.5x
- 回撤 > 25%（低位启动）
- RPS20 前30%

### 科创板（成长股）
- 研发投入 ≥ 10%
- 营收增速 ≥ 30%
- 国产替代名单
- 渗透率 ≤ 30%

### ETF（轮动）
- 溢价率 > 0.5% → 卖出信号
- 折价率 > 0.5% → 买入信号
- 板块指数涨幅 > 3%

### 基金（选基）
- 超额收益 ≥ 5%
- 最大回撤 ≤ -20%
- 经理从业 ≥ 5年
- 规模 10-100亿

## 风控统一规则（v4.0-2026-05-16）

| 市场温度 | 仓位上限 | 操作限制 |
|---------|---------|---------|
| ≥80℃（过热） | 0-10% | 只卖不买，逐步减仓 |
| 70-80℃（高温） | 20% | P0禁止新开仓 |
| 50-70℃（正常） | 50% | 正常操作，严选信号 |
| 35-50℃（偏弱） | 30% | 只做强势股，轻仓 |
| 25-35℃（冰点） | 10% | 🚫 R18禁止新开仓 |
| <25℃（恐慌） | 0% | 🚨 V10强制空仓 |

### 新增执行规则摘要

| 规则 | 说明 |
|------|------|
| R18 | 冰点期入场禁止：温度<35℃禁止任何新开仓 |
| R19 | 大盘趋势确认：趋势+赚钱效应+板块联动三重检查 |
| R20 | 浮亏分级响应：0-2%绿→2-4%黄→4%~止损橙→止损红 |
| R21 | 批量建仓冷却：单日≤2只，同板块≤30%，间隔≥30min |
| V10 | 恐慌环境：恐慌<25禁止操作，<15强制空仓 |
