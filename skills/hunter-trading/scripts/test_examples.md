## 六、支付流程测试

### 测试8：支付确认

**输入**：
```
支付0.05元查看详细分析
```

**预期流程**：
1. 显示支付金额确认
2. 生成支付宝收款码/链接
3. 用户完成支付
4. 解锁完整报告

**预期输出（支付后）**：
```
✅ 支付成功

正在解锁完整报告...
[显示完整分析报告]
```

## 七、边界测试

### 测试9：无效股票代码

**输入**：
```
分析 000000
```

**预期输出**：
```
❌ 未找到股票 000000

请检查股票代码是否正确，A股代码为6位数字。
示例：600519（贵州茅台）、000001（平安银行）
```

### 测试10：非A股股票

**输入**：
```
分析 AAPL
```

**预期输出**：
```
❌ 暂不支持分析 AAPL

本Skill仅支持沪深A股市场。
支持的股票示例：
- 上证：600519（贵州茅台）、600036（招商银行）
- 深证：000001（平安银行）、300750（宁德时代）
```
