# 猎手交易分析Skill

> A股短线交易智能分析，部署到龙虾(OpenClaw)平台

---

## 概述

猎手交易分析Skill将猎手模拟交易系统的核心能力封装为可独立运行的技能模块，支持股票查询、技术分析、情绪周期判断、仓位管理和风险控制等功能。

### 核心能力

- **技术分析**：基于R35/R38规则的低位启动信号和连板龙头识别
- **情绪周期**：恐贪指数监控和五阶段仓位管理
- **风险控制**：止损规则、主力诱多识别、事件驱动风控
- **支付集成**：支付宝AI收，按次付费（¥0.05/次）

## 快速开始

### 安装

```bash
# 1. 创建Skill目录
mkdir -p ~/.openclaw/skills/hunter-trading

# 2. 复制文件
cp -r ./猎手Skill/* ~/.openclaw/skills/hunter-trading/

# 3. 配置支付（编辑config/payment.yaml填入AppID）
nano ~/.openclaw/skills/hunter-trading/config/payment.yaml
```

### 使用

```
# 基本语法
分析 <股票代码或名称>     # 完整分析报告
查询 <股票代码或名称>     # 快速查询
诊断 <持仓列表>          # 持仓诊断
止损 <股票代码/成本>      # 止损计算
仓位                      # 市场仓位建议
```

## 核心规则

### R系技术规则

| 规则 | 触发条件 | 操作建议 |
|-----|---------|---------|
| R35 | 回撤>25% + 量比>1.5x + 涨幅>7% | 关注买入机会 |
| R38 | 连板≥2天 + 板块涨停≥3家 | 持有或追龙头 |

### R系风控规则

| 规则 | 触发条件 | 操作建议 |
|-----|---------|---------|
| R1 | 指数接近±0.5%关口 | 观察突破质量 |
| R2 | 涨跌比决定仓位上限 | 2:1以上可重仓 |
| R3 | 指数涨但家数少 | 立即减仓至3成 |
| R4 | 事件日前3天 | 禁止买入 |

### R系情绪规则

| 规则 | 触发条件 | 操作建议 |
|-----|---------|---------|
| R6 | 恐贪指数≥85分 | 仓位≤2成或空仓 |
| R7 | 涨停家数判断 | 五阶段仓位管理 |
