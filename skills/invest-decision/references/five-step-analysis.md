# 股票五步法分析 — 猎手系统增强版

> 基于长投学堂五步分析法 + 猎手市场情绪与资金流体系，适用于A股/港股/美股

---

## 五步法概览

```
【前置】市场环境评估  →  涨停家数/涨跌比/主力净流向
Step 1: 好公司       →  净利润真实性 + 七大竞争力
Step 2: 好未来       →  行业天花板 + 增速预测
Step 3: 好价格       →  PE/PB百分位 + 绝对估值
Step 4: 好买卖       →  分档建仓 + 卖出逻辑
Step 5: 风险提示      →  行业/公司/估值风险 + 逆向排雷
```

## 【前置】市场环境评估 — 猎手入场门槛

> ⛔ 市场环境不合格时，基本面再好也不进场

### 核心指标

| 指标 | 合格线 | 不合格信号 | 数据来源 |
|------|--------|-----------|---------|
| 涨停家数 | ≥ 30家 | < 20家（冰点）| akshare `stock_zt_pool_em` |
| 涨跌比 | > 1:1 | < 1:2（极差）| akshare 获取上涨/下跌家数 |
| 主力净流向 | 连续2日净流入 | 龙头跌停 + 主力净流出 | akshare `stock_individual_fund_flow` |
| 市场主线 | 有明确主线 | 无主线/杂乱 | 盘面观察 |

### 资金流向分析

**三日主力净流向法**：
```
净流向 = 主力净流入（超大单+大单）
判断标准：
  + 连续3日净流入 → 机构建仓信号 🟢
  + 1-2日净流入后净流出 → 背离预警 🟡
  - 连续3日净流出 → 机构撤退 🔴
```

**背离预警（高危）**：
- 基本面龙头出现跌停 + 主力净流出 = 机构出货
- 不可因「基本面好」而忽视此信号

### 市场情绪评级

| 评级 | 涨停家数 | 涨跌比 | 操作 |
|------|---------|--------|------|
| 🟢 活跃 | ≥ 50家 | > 2:1 | 积极做多，可追强势股 |
| 🟡 正常 | 30-50家 | 1:1~2:1 | 轻仓试探，等待主线 |
| 🔴 冰点 | 10-30家 | < 1:3 | 禁止追涨，只做防御 |
| ⚫ 极差 | < 10家 | 1:4+ | 空仓休息，不抄底 |

### 猎手入场决策表

| 市场情绪 | 资金流向 | 操作 |
|---------|---------|------|
| 🟢 活跃 | 主力净流入 | 可以建仓，跟随主线 |
| 🟡 正常 | 主力净流入 | 轻仓（<30%），快进快出 |
| 🟡 正常 | 主力净流出 | 禁止买入，等信号 |
| 🔴 冰点 | 任意 | 空仓，不接飞刀 |
| ⚫ 极差 | 任意 | 完全休息 |

## Step 1：好公司

### 1.1 净利润真实性验证

**核心指标**：
- 扣非净利润 ≠ 净利润（扣除非经常性损益）
- 经营现金流净额 vs 净利润（>1 说明利润质量高）
- 应收帐款增速 vs 营收增速（应收增速不应大幅超过营收）

**交叉验证**：
- 年报数据 vs 季报数据趋势是否一致
- 同业对比：毛利率、净利率是否异常

### 1.2 七大竞争力分析

| # | 竞争力维度 | 评估问题 | 量化指标 |
|---|---------|---------|---------|
| 1 | 定价权 | 能否提价而不失去客户？ | 毛利率趋势 |
| 2 | 重复购买 | 客户会重复购买吗？ | 客户留存率 |
| 3 | 网络效应 | 用户越多价值越高？ | 用户增速/市占率 |
| 4 | 低成本护城河 | 能以更低成本竞争？ | 三费占比 |
| 5 | 监管壁垒 | 受到政策保护？ | 牌照/资质 |
| 6 | 品牌护城河 | 品牌能带来溢价？ | 品牌溢价率 |
| 7 | 资本门槛 | 新进入者需要大量资本？ | 资本支出占比 |

## Step 2：好未来

### 2.1 行业天花板分析

**TAM/SAM/SOM三层模型**：
- TAM（总可触达市场）：行业整体规模
- SAM（可服务市场）：公司能触及的市场
- SOM（可获得市场）：短期能拿下的份额

### 2.2 成长性预测

```
未来3年增速 = 行业增速 × 市占率提升 × 单客户价值提升
```

## Step 3：好价格

### 3.1 PE/PB百分位法

| 百分位 | 信号 | 操作 |
|--------|------|------|
| <20% | 严重低估 | 强烈买入 |
| 20-40% | 低估 | 买入 |
| 40-60% | 合理 | 持有/观望 |
| 60-80% | 高估 | 减仓 |
| >80% | 严重高估 | 卖出 |

### 3.2 安全边际决策

| 估值方法 | 结果 | 决策 |
|---------|------|------|
| PE百分位 < 30% 且 DCF折价 > 30% | 双重低估 | 强烈买入 |
| PE百分位 < 30% 或 DCF折价 > 30% | 单重低估 | 买入 |
| PE百分位 30-70% | 合理区间 | 持有/观望 |
| PE百分位 > 70% | 高估 | 卖出或减仓 |

---

## Step 4：好买卖

### 4.1 分档建仓策略

```
保守价格（DCF内在价值×0.7）→ 首仓30%
合理价格（DCF内在价值×0.85）→ 加仓30%
合理偏低（DCF内在价值×1.0）→ 末仓40%
```

### 4.2 卖出逻辑

**强制卖出条件**：
1. 基本面恶化：连续2年营收负增长
2. 估值极端：PE百分位>90%且明显透支3年业绩
3. 发现更优标的：新标的预期收益率高20%以上
4. 市场情绪逆转：涨停家数从>30骤降至<20

---

## Step 5：风险提示

### 5.1 市场情绪风险

| 风险类型 | 评估维度 | 预警信号 |
|---------|---------|---------|
| 冰点风险 | 涨停家数 | 连续3日<20家 |
| 情绪逆转 | 涨跌比 | 从>2:1骤降至<1:2 |
| 资金撤退 | 主力净流向 | 龙头股主力连续净流出 |

### 5.2 公司风险

| 风险类型 | 观察指标 | 预警信号 |
|---------|---------|---------|
| 经营风险 | 现金流 | 经营现金流连续2年为负 |
| 财务风险 | 资产负债率 | >70%为高负债 |
| 治理风险 | 大股东行为 | 关联交易异常 |
| 背离风险 | 资金流向 | 基本面好但主力净流出 |

### 5.3 猎手逆向排雷清单

**🚨 触发任意一条，禁止买入**：
- [ ] 涨停家数 < 20家（市场冰点）
- [ ] 涨跌比 < 1:3（亏钱效应极差）
- [ ] 目标个股主力净流出 + 龙头跌停
- [ ] 基本面好但资金连续3日净流出（背离）
- [ ] 市场无明确主线

**⚠️ 触发任意一条，谨慎操作**：
- [ ] 涨停家数 20-30家（冰点边缘）
- [ ] 目标个股所在板块主力净流出
- [ ] 前龙头出现跌停（板块轮动风险）
