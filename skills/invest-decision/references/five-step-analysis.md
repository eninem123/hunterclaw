# 股票五步法分析 — 长投学堂体系

> 基于长投学堂五步分析法，适用于A股/港股/美股

---

## 五步法概览

```
Step 1: 好公司   → 净利润真实性 + 七大竞争力
Step 2: 好未来   → 行业天花板 + 增速预测
Step 3: 好价格   → PE/PB百分位 + 绝对估值
Step 4: 好买卖   → 分档建仓 + 卖出逻辑
Step 5: 风险提示  → 行业/公司/估值风险
```

---

## Step 1：好公司

### 1.1 净利润真实性验证

**核心指标**：
- 扣非净利润 ≠ 净利润（扣除非经常性损益）
- 经营现金流净额 vs 净利润（>1 说明利润质量高）
- 应收帐款增速 vs 营收增速（应收增速不应大幅超过营收）

**交叉验证**：
- 年报数据 vs 季报数据趋势是否一致
- 同业对比：毛利率、净利率是否异常
- 审计意见：非标审计报告需警惕

**数据来源**：公司年报（巨潮资讯网）、券商研报、同花顺iFinD

### 1.2 七大竞争力分析

| # | 竞争力维度 | 评估问题 | 量化指标 |
|---|---------|---------|---------|
| 1 | 定价权 | 能否提价而不失去客户？ | 毛利率趋势 |
| 2 | 重复购买 | 客户会重复购买吗？ | 客户留存率 |
| 3 | 网络效应 | 用户越多价值越高？ | 用户增速/市占率 |
| 4 | 低成本护城河 | 能以更低成本竞争？ | 三费占比 |
| 5 | 监管壁垒 | 受到政策保护？ | 牌照/资质 |
| 6 | 品牌护城河 | 品牌能带来溢价？ | 品牌溢价率 |
| 7 | 资本门槛 | 新进入者需要大量资本？ | 资本支出占比 |

**护城河打分**（5分制）：
- 5分：多条护城河，竞争优势显著
- 3-4分：1-2条护城河，持续竞争力
- 1-2分：竞争激烈，护城河薄弱

## Step 2：好未来

### 2.1 行业天花板分析

**TAM/SAM/SOM三层模型**：
- TAM（总可触达市场）：行业整体规模
- SAM（可服务市场）：公司能触及的市场
- SOM（可获得市场）：短期能拿下的份额

**评估方法**：
- 行业增速：>10%为高速增长，5-10%为稳定增长，<5%为低速/衰退
- 市场集中度：CR3/CR5是否在提升（提升=龙头受益）
- 政策方向：支持/限制/中性

### 2.2 成长性预测

**增速预测框架**：
```
未来3年增速 = 行业增速 × 市占率提升 × 单客户价值提升
```

**关键财务指标**：
- 营收增速：>20%为高成长，10-20%为稳健
- 净利润增速：应与营收增速匹配（说明增长质量）
- 毛利率趋势：平稳或上升=好，下降=坏

**风险信号**：
- 营收增长靠并购，剥离后内生增速如何？
- 产能扩张是否匹配市场需求？
- 研发投入是否持续稳定？

## Step 3：好价格

### 3.1 PE/PB百分位法

**A股历史百分位**（近10年）：
| 百分位 | 信号 |
|--------|------|
| <20% | 严重低估 ⚡ |
| 20-40% | 低估 |
| 40-60% | 合理 |
| 60-80% | 高估 |
| >80% | 严重高估 🔥 |

**使用注意**：
- 强周期股（券商/钢铁/煤炭）PE失真，用PB
- 成长股PE高是正常的，需结合增速
- 金融股用PB为主（银行破净常见）

### 3.2 绝对估值法

**DCF三阶段模型**：
```
内在价值 = Σ(FCF_t / (1+r)^t) + 终值/(1+r)^n
```
其中：FCF=自由现金流，r=折现率（通常用无风险利率+风险溢价），n=显性期年数

**简化估值**：
- 永续年金：FCF / (折现率 - 永续增长率)
- 保守假设：折现率=10%，永续增长率=3%

**PEG估值**：
```
PEG = PE / (营收增速×100)
< 1 = 相对便宜；> 1.5 = 相对昂贵
```

### 3.3 估值决策表

| 估值方法 | 结果 | 决策 |
|---------|------|------|
| PE百分位 < 30% 且 DCF折价 > 30% | 双重低估 | 强烈买入 |
| PE百分位 < 30% 或 DCF折价 > 30% | 单重低估 | 买入 |
| PE百分位 30-70% | 合理区间 | 持有/观望 |
| PE百分位 > 70% | 高估 | 卖出或减仓 |

## Step 4：好买卖

### 4.1 分档建仓策略

**三档建仓法**（以安全边际为中心）：
```
保守价格（DCF内在价值×0.7）→ 首仓30%
合理价格（DCF内在价值×0.85）→ 加仓30%
合理偏低（DCF内在价值×1.0）→ 末仓40%
```

**仓位管理原则**：
- 单只股票仓位上限：20%（高度确信可30%）
- 行业集中度上限：40%
- 持仓数量：5-10只为适度集中

### 4.2 卖出逻辑

**强制卖出条件**（触发即卖）：
1. 基本面恶化：连续2年营收负增长
2. 估值极端：PE百分位>90%且明显透支3年业绩
3. 发现更优标的：新标的预期收益率高20%以上

**持有条件**：
1. 核心竞争力未被削弱
2. 估值未进入极端高估区
3. 行业基本面未发生根本性变化

## Step 5：风险提示

### 5.1 行业风险

| 风险类型 | 评估维度 | 预警信号 |
|---------|---------|---------|
| 政策风险 | 行业受监管程度 | 政策转向、限制性文件 |
| 周期风险 | 强/弱周期 |宏观指标下行 |
| 技术风险 | 技术路线颠覆 | 新技术替代 |
| 需求风险 | 需求是否稳定 | 人口/消费习惯变化 |

### 5.2 公司风险

| 风险类型 | 观察指标 | 预警信号 |
|---------|---------|---------|
| 经营风险 | 现金流 | 经营现金流连续2年为负 |
| 财务风险 | 资产负债率 | >70%为高负债 |
| 治理风险 | 大股东行为 | 关联交易异常 |
| 依赖风险 | 客户/供应商集中度 | 第一大客户占比>30% |

### 5.3 估值风险

| 风险类型 | 评估方法 | 预警信号 |
|---------|---------|---------|
| 增长不及预期 | 实际增速 vs 预测 | 连续2年低于预测30% |
| 利率风险 | DCF对利率敏感性 | 利率上升>2%对估值影响 |
| 流动性风险 | 日均成交额 | 日均<1000万为低流动性 |
