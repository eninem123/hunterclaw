# 全链路商业情报与投资决策系统

> 政策解读 → 商机挖掘 → 逆向风控 → 标的筛选

---

## 四阶段分析框架

```
阶段1: 去伪存真与政策解码（政策三看模型）
阶段2: 技术风口验证（Gartner曲线 + 市场规模）
阶段3: 消费趋势研判（人/货/场框架）
阶段4: 上市公司财务量化（杜邦拆解 + 估值锚定）
```

## 阶段1：政策三看模型

### 1.1 看原文（硬指标）

**5W1H解构政策**：
- What（什么）：政策的直接目标是什么？
- Why（为什么）：出台背景和动机？
- Who（谁受益）：政策的对象是谁？
- Where（哪里）：试点/全国/特定区域？
- When（何时）：实施时间节点？
- How（如何执行）：配套措施和资金来源？

**核心判断**：
- 硬指标（有明确数字）：规模目标、补贴金额、减税幅度
- 软目标（模糊表述）："大力发展"、"稳步推进"——利好有限

### 1.2 看落地（配套文件）

**三个维度评估落地可行性**：
1. **资金来源**：中央财政/地方财政/社会资本？预算是否充足？
2. **执行机构**：哪个部委牵头？地方执行能力如何？
3. **时间表**：政策目标的时间节点是否合理？

**落地进度信号**：
- 配套文件发布频率（3个月内出细则=执行力强）
- 试点城市/企业名单公布
- 补贴拨付进度

### 1.3 看受益标的

**三层穿透法**：
```
直接受益层 → 政策直接扶持的企业/行业
核心受益层 → 产业链中不可替代的环节
传导受益层 → 间接受益的上游/互补品
```

**评估框架**：
| 层级 | 特征 | 投资逻辑 |
|------|------|---------|
| 直接层 | 政策明确支持 | 短期爆发力强，持续性取决于政策力度 |
| 核心层 | 护城河深，不可替代 | 中期确定性高，估值有支撑 |
| 传导层 | 间接受益，轮动特点 | 行情晚于直接层，需要跟踪验证 |

## 阶段2：Gartner曲线阶段判断

### 2.1 技术成熟度五阶段

```
[技术萌芽期] → [期望膨胀期] → [泡沫破裂期] → [稳步爬升期] → [生产成熟期]
     ↑              ↑              ↑              ↑              ↑
   概念期         狂热期         低迷期         复苏期        成熟期
```

**阶段特征与投资策略**：

| 阶段 | 标志特征 | 投资策略 |
|------|---------|---------|
| 萌芽期 | 实验室阶段，无商业化 | 小仓位布局纯概念，不追高 |
| 膨胀期 | 媒体狂热，概念股普涨 | 区分"假妖怪"和"真龙头"，只买有实质业务的 |
| 泡沫期 | 业绩证伪，估值崩塌 | 坚决回避，不接飞刀 |
| 爬升期 | 技术成熟，应用落地 | 选择已经实现盈亏平衡的公司 |
| 成熟期 | 行业格局稳定 | 买龙头等分红，不追求高增长 |

### 2.2 市场规模三维验证

**TAM/SAM/SOM三层验证**：
```
TAM（潜在市场总量）：行业终极规模的理论上限
SAM（可服务市场）：公司能触及的地域/客群范围
SOM（可获得市场）：3年内实际能拿到的份额
```

**三维评估矩阵**：
| 维度 | 问题 | 乐观/悲观/现实 |
|------|------|--------------|
| 渗透率 | 当前多少？ | <10%=低，10-50%=中，>50%=高 |
| 增速 | CAGR多少？ | >20%=高速，10-20%=稳健 |
| 集中度 | 龙头份额变化？ | 提升=龙头受益，下降=竞争激烈 |

## 阶段3：消费趋势研判

### 3.1 人/货/场框架

**人（消费者）**：
- 人口结构：Z世代/银发族/中产
- 消费观念：性价比/体验/健康/国潮
- 信号强度：社媒讨论热度、搜索指数变化

**货（产品）**：
- 产品周期：导入/成长/成熟/衰退
- 创新程度：颠覆式创新/渐进式创新
- 替代威胁：是否有替代品正在崛起

**场（渠道）**：
- 渠道变革：线上化率/私域/直播
- 地理扩张：下沉市场/海外市场
- 终端变化：新型门店/自动售货/社区团购

### 3.2 信号强度分级

| 等级 | 信号 | 含义 |
|------|------|------|
| 🟢 强信号 | 多个独立数据源验证 | 趋势确立，可重仓 |
| 🟡 中信号 | 单一数据源确认 | 需跟踪验证，谨慎 |
| 🔴 弱信号 | 仅有媒体报道/传言 | 不作为投资依据 |

## 阶段4：财务量化分析

### 4.1 杜邦分析拆解

```
ROE = 净利率 × 资产周转率 × 权益乘数
     = (净利润/营收) × (营收/总资产) × (总资产/净资产)
```

**三维度打分**：
| 指标 | 计算 | 评估 |
|------|------|------|
| 净利率 | 净利润/营收 | >15%=优秀，5-15%=良好，<5%=一般 |
| 资产周转率 | 营收/总资产 | >1=高效，0.5-1=正常，<0.5=低效 |
| 权益乘数 | 总资产/净资产 | <3=稳健，3-5=中性，>5=高杠杆 |

**ROE驱动因素识别**：
- 高净利率+低周转：好生意（品牌/专利）
- 低净利率+高周转：苦生意（零售/贸易）
- 高杠杆放大：需要警惕

### 4.2 同业对比

**关键指标对比表**：

|------|------|------|----------|

### 4.3 估值锚定

**绝对估值（DCF简化版）**：
```
内在价值 = FCF / (WACC - 永续增长率)
WACC = 10%（无风险利率+风险溢价）
永续增长率 = 3%（保守假设）
```

**相对估值锚定**：
- PE：历史分位 vs 行业分位（取较低者）
- PB：破净=极度悲观（可能是买入机会）
- PS：亏损企业适用，PS<10=合理

## 逆向排雷机制

### 雷区清单

**财务报表雷**：
- [ ] 经营现金流连续2年<0
- [ ] 应收账款增速持续>营收增速20%以上
- [ ] 存货突然大幅增加但营收不增
- [ ] 固定资产激增但产能利用率低
- [ ] 关联交易异常（大股东体外循环）

**公司治理雷**：
- [ ] 大股东高比例质押（>50%）
- [ ] 商誉占净资产>50%
- [ ] 频繁更换审计机构
- [ ] 高管层频繁离职
- [ ] 实际控制人变更

**行业风险雷**：
- [ ] 政策转向打压
- [ ] 核心产品被列入负面清单
- [ ] 替代技术路线出现
- [ ] 行业产能严重过剩

### 排雷决策

| 雷区类型 | 危险等级 | 处理方式 |
|---------|---------|---------|
| 现金流断裂风险 | 🔴极高 | 立即卖出 |
| 大股东质押爆仓 | 🔴极高 | 回避 |
| 政策打压行业 | 🔴极高 | 坚决卖出 |
| 应收账款异常 | 🟡中等 | 深入调查后决策 |
| 商誉减值风险 | 🟡中等 | 谨慎，预留安全边际 |
| 高管离职 | 🟢低 | 观察，持续跟踪 |

## 操作四象限

| 象限 | 条件 | 操作 |
|------|------|------|
| **高确信+低风险** | 基本面优秀+估值合理+政策支持 | 重仓持有 |
| **高确信+高风险** | 基本面优秀+估值偏高 | 轻仓，等待回调 |
| **低确信+低风险** | 基本面一般+估值低位 | 定投，积累筹码 |
| **低确信+高风险** | 基本面差+估值高 | 回避 |
