# 投资决策助手

整合三大投资分析体系，为投资者提供从理念到实操的全链路支持。

## 核心模块

| 模块 | 功能 | 适用场景 |
|------|------|---------|
| 巴菲特投资智慧 | 77条原则查询、主题检索、随机推荐 | 投资理念学习、决策参考 |
| 五步法分析 | 个股深度基本面分析 | 个股研究、选股 |
| 商业情报分析 | 政策解读、行业判断、风险排雷 | 财经新闻解读、宏观决策 |

## 模块一：巴菲特投资智慧

### 核心能力

**查询方式**：
```
// 按主题查询（10大主题：长期投资/风险管理/价值评估/企业质量/投资心理/买入原则/市场心理/能力圈/人生智慧/安全边际）
→ 直接说主题或关键词，如"安全边际"、"长期投资"、"风险管理"

// 随机推荐一条原则
→ "随机推荐一条巴菲特原则"

// 查询特定原则编号
→ "巴菲特原则#77"
```

**主题关键词映射**：
- 长期投资：复利、耐心、持有十年、频繁交易
- 风险管理：分散、止损、本金安全、杠杆
- 价值评估：内在价值、DCF、估值、ROE
- 企业质量：护城河、竞争力、七大竞争力
- 投资心理：贪婪、恐惧、逆向、心理
- 买入原则：买入时机、建仓、安全边际
- 市场心理：市场先生、投票机、称重机、逆向
- 能力圈：懂不懂、能力边界、不懂不做
- 人生智慧：学习、自知
- 安全边际：折扣、低价买入、缓冲

**决策应用流程**：
1. 用户询问某股票是否值得买入 → 先进行五步法分析
2. 分析过程中引用相关巴菲特原则强化逻辑
3. 给出结论时用巴菲特原则作为决策依据

### 示例对话

```
用户：这股票能买吗？
→ 使用五步法分析该股票

   这只股票需要你确认是否具备长期持有条件..."
```

## 模块二：五步法个股分析

### 五步分析流程

```
Step 1: 好公司 → 净利润真实性 + 七大竞争力
Step 2: 好未来 → 行业天花板 + 增速预测
Step 3: 好价格 → PE/PB百分位 + DCF估值
Step 4: 好买卖 → 分档建仓策略 + 卖出逻辑
Step 5: 风险提示 → 行业/公司/估值风险
```

### 数据获取

**自动获取（按优先级）**：
1. 通过 ths-financial-data 或 akshare-finance 获取实时行情、财务数据
2. 通过 eastmoney-tools 获取研报、新闻
3. 历史百分位数据需结合 iFinD/akshare

**触发词**：
- "分析[股票代码/名称]"
- "个股分析"
- "股票研究"
- "五步法"
- "帮我看看[股票]能不能买"

### 五步法详解

#### Step 1 好公司

**净利润真实性**：
- 扣非净利润 vs 净利润（差距大需警惕）
- 经营现金流/净利润 > 1 = 利润质量好
- 应收增速 vs 营收增速（应收增速不应大幅超过营收）

**七大竞争力**（5分制）：
1. 定价权——能提价吗？
2. 重复购买——客户粘性？
3. 网络效应——用户越多越值钱？
4. 低成本护城河——能以更低成本竞争？
5. 监管壁垒——有牌照/资质保护？
6. 品牌护城河——品牌带来溢价？
7. 资本门槛——新进入者需要大量资本？

**护城河评估**：
- 5分：多条护城河，竞争优势显著
- 3-4分：1-2条护城河，持续竞争力
- 1-2分：竞争激烈，护城河薄弱

#### Step 2 好未来

**行业天花板**（TAM/SAM/SOM）：
- 行业增速：>10%高速，5-10%稳健，<5%低速
- 市场集中度：CR3/CR5提升=龙头受益

**增速预测**：
```
未来3年增速 = 行业增速 × 市占率提升 × 单客户价值提升
```

**关键指标**：营收增速>20%高成长；净利润增速与营收增速匹配=增长质量好

#### Step 3 好价格

**PE/PB百分位**（A股近10年）：
| 百分位 | 信号 | 操作 |
|--------|------|------|
| <20% | 严重低估 | 强烈买入 |
| 20-40% | 低估 | 买入 |
| 40-60% | 合理 | 持有/观望 |
| 60-80% | 高估 | 减仓 |
| >80% | 严重高估 | 卖出 |

**DCF简化估值**：
```
内在价值 = FCF / (10% - 3%) = FCF / 7%
永续增长率保守假设 = 3%
折现率 = 10%
```

**安全边际决策**：
- 双重低估（PE<20% 且 DCF折价>30%）= 强烈买入
- 单重低估 = 买入
- 合理区间 = 持有
- 高估 = 减仓/卖出

#### Step 4 好买卖

**三档建仓法**：
```
保守价格（DCF内在价值×0.7）→ 首仓30%
合理价格（DCF内在价值×0.85）→ 加仓30%
合理偏低（DCF内在价值×1.0）→ 末仓40%
```

**仓位原则**：
- 单只上限：20%（高度确信可30%）
- 行业上限：40%
- 持仓数量：5-10只

**强制卖出条件**：
1. 基本面恶化：连续2年营收负增长
2. 估值极端：PE百分位>90%且透支3年业绩
3. 发现更优标的：预期收益率高20%以上

#### Step 5 风险提示

**行业风险**：政策转向、周期风险、技术颠覆、需求变化

**公司风险**：现金流断裂、资产负债率>70%、治理问题、大股东高比例质押

**估值风险**：增长不及预期、利率上升敏感性、流动性枯竭

## 模块三：全链路商业情报分析

### 四阶段分析链

```
阶段1: 政策解码（政策三看模型）
阶段2: 商机挖掘（Gartner曲线 + 市场规模）
阶段3: 逆向风控（排雷清单）
阶段4: 标的筛选（三层穿透法）
```

### 政策三看模型

**一看原文**（硬指标）：
- 有明确数字的指标（补贴金额、规模目标）= 硬指标
- "大力发展"类模糊表述 = 软目标，利好吃有限

**二看落地**：
- 3个月内出配套细则 = 落地执行力强
- 试点城市/企业名单公布
- 补贴拨付进度

**三看受益标的**：
- 直接受益层 → 政策直接扶持（短期爆发）
- 核心受益层 → 产业链不可替代环节（中期确定性）
- 传导受益层 → 间接受益（轮动晚于直接层）

### Gartner曲线阶段

| 阶段 | 特征 | 策略 |
|------|------|------|
| 萌芽期 | 实验室，无商业化 | 小仓位布局概念 |
| 膨胀期 | 媒体狂热，概念普涨 | 区分真假，只买有实质业务的 |
| 泡沫期 | 业绩证伪，估值崩塌 | 坚决回避 |
| 爬升期 | 技术成熟，应用落地 | 选已盈亏平衡的公司 |
| 成熟期 | 格局稳定 | 买龙头等分红 |

### 三层穿透法选股

**直接层**：政策直接扶持行业内的龙头
**核心层**：产业链关键环节（技术壁垒高、不可替代）
**传导层**：上游原材料/设备、下游渠道

### 逆向排雷清单

**财务报表雷（立即回避）**：
- 经营现金流连续2年<0
- 应收增速持续>营收增速20%
- 存货突增但营收不增
- 大股东关联交易异常

**公司治理雷**：
- 大股东质押>50%
- 商誉占净资产>50%
- 频繁更换审计机构
- 高管层频繁离职

**行业风险雷**：
- 政策明确打压方向
- 替代技术路线出现
- 行业产能严重过剩

1. **实时行情**：ths-financial-data 或 akshare-finance

## 注意事项

- 本技能输出仅供参考，不构成投资建议
- 股市有风险，投资需谨慎
- 任何分析方法都不能保证盈利
- 建议配合仓位管理和风险分散
- 历史百分位数据仅供参考，不代表未来表现
