```python
class DataAnalyzer:
    """数据分析器"""
    
    def calculate_ma(self, prices, period) -> float
    def calculate_rsi(self, prices, period=14) -> float
    def calculate_macd(self, prices) -> Tuple[float, float, float]
    def calculate_bollinger_bands(self, prices) -> Tuple[float, float, float]
    def calculate_kdj(self, highs, lows, closes) -> Tuple[float, float, float]
    def identify_trend(self, ma5, ma10, ma20, ma60) -> Trend
    def calculate_signal(self, indicators, price, chg_pct, volume_ratio) -> SignalStrength
    def analyze_stock(self, code, name, kline_data, ...) -> AnalysisResult
    def batch_analyze(self, stocks_data) -> List[AnalysisResult]
    def get_top_candidates(self, results, min_score, max_count) -> List[AnalysisResult]
```

**使用示例：**

```python
from data_analyzer import DataAnalyzer

# 创建分析器
analyzer = DataAnalyzer()

# 分析单只股票
result = analyzer.analyze_stock(
    code='000001',
    name='平安银行',
    kline_data=kline_data,
    current_price=10.8,
    chg_pct=2.5,
    volume_ratio=1.5
)

print(f"趋势: {result.trend.value}")
print(f"信号: {result.signal.value}")
print(f"评分: {result.score:.1f}")
print(f"理由: {', '.join(result.reasons)}")

# 批量分析
results = analyzer.batch_analyze(stocks_data)
top_stocks = analyzer.get_top_candidates(results, min_score=70, max_count=5)
```

### 1. 新增核心规则（R18-R21）

#### R18: 主力资金异常回流识别规则

```
【核心问题】主力流出后异常回流，是真实反转还是诱多陷阱？

【识别框架】
异常回流判定条件（同时满足才视为有效）：
□ 连续3日主力净流出后，单日净流入 > 前3日累计流出的50%
□ 净流入当日成交额较前5日均值萎缩 < 20%（放量才可信）
□ 当日指数涨幅 > 1% 且 涨跌比 > 1.5:1

【操作规则】
1. 有效回流 → 可轻仓跟随（1-2成）
2. 无效回流（放量小涨/缩量大涨）→ 视为诱多，不操作
3. 疑似回流 → 等待次日确认，不追高

【风险预警】
- 连续2日净流入量递减 → 回流动能衰减 → 减仓
- 回流后第3日再度净流出 → 确认诱多 → 清仓
```

#### R19: 板块内部强弱分化识别规则

```
【核心问题】同板块内部分化，龙头涨但跟风跌，如何识别板块真正方向？

【强弱分化指数】
分化指数 = 板块内涨幅 > 3%的股票数量 / 跌幅 > 3%的股票数量
- > 2.0：龙头领涨，跟风健康 → 板块强势
- 1.0 ~ 2.0：正常分化
- 0.5 ~ 1.0：跟风乏力 → 警惕退潮
- < 0.5：龙头孤涨，板块退潮 → 减仓

【操作规则】
1. 分化指数 > 2.0 → 板块强势，可加仓龙头
2. 分化指数 < 0.5 → 板块退潮，减仓/清仓
3. 连续3日分化指数 > 1.5 → 可能从发酵期过渡到高潮期，逐步止盈

【交叉验证】
- V05板块净流入率 + 分化指数 → 双重确认
- 分化指数下行 + V05净流入率下降 → 提早于板块跌幅出局
```

#### R20: 高位缩量加速识别规则

```
【核心问题】高位缩量加速上涨，看着强势实则风险累积。

【量化定义】
高位缩量加速 = 连续2日涨幅 > 3% + 成交额低于5日均值30%以上

【操作规则】
| 情况 | 操作 |
|------|------|
| 首次出现缩量加速 | 持有但设置成本+3%移动止盈 |
| 连续2日缩量加速 | 减仓50%，剩余仓位保本价 |
| 缩量加速后放量下跌 | 立即清仓，不抱幻想 |
| 缩量加速后突破放量 | 加回减仓部分，但总仓位不超原水平 |

【配合V07 ETF资金流向】
- ETF同步缩量 → 主线降温信号
- ETF放量但个股缩量 → 资金从个股转移到ETF，警惕抽血效应
```

#### R21: 市场冰封/解封系统规则

```
【核心问题】极端行情下靠主观判断是否操作，缺少量化触发/解除标准。

【冰封条件】（任一触发即进入冰封期）
□ 沪深300指数单日跌幅 > 3%
□ 涨停家数 < 15家 且 跌停家数 > 50家
□ 主力资金单日净流出 > 500亿
□ 涨跌比 < 1:3（涨<500家，跌>1500家）
□ 期指基差（V06）全部合约贴水 > 0.5%

【冰封期规则】
1. 禁止一切新买入操作
2. 持仓只执行止损，不执行止盈
3. 总仓位强制降至2成以下
4. 冰封期不参与任何短线/抄底操作
5. 冰封期持续至少1个完整交易日

【解封条件】（全部满足才解除）
□ 沪深300指数连续2日不再创新低
□ 涨停家数回升至30家以上
□ 主力资金开始净流入（单日>100亿）
□ 涨跌比恢复到1:1以上
□ V05板块净流入率前3板块中有2个净流入率 > 1%

【冰封/解封对照】
| 维度 | 冰封条件（任一触发） | 解封条件（全部满足） |
|------|---------------------|---------------------|
| 指数 | 沪深300单日跌 > 3% | 连续2日不再创新低 |
| 涨停 | 涨停<15家，跌停>50家 | 涨停回升至30家以上 |
| 资金 | 主力净流出 > 500亿 | 主力开始净流入>100亿 |
| 涨跌比 | < 1:3 | 恢复至1:1以上 |
| 期指信号 | V06全部贴水>0.5% | V05有2个板块净流入率>1% |

【冰封期解除后操作】
1. 解封首日：仅恢复观察，不操作
2. 解封次日：轻仓试探（1-2成）
3. 解封后第3日：确认市场企稳后恢复标准操作
```

### 2. 市场温度仓位管理表（R7/R8 升级版）

> 融合R7情绪周期、R8情绪极值量化、R2赚钱效应系数的统一仓位管理表。

#### 市场温度等级

| 温度等级 | 综合评分 | 对应情绪周期 | 仓位上限 | 核心策略 |
|----------|----------|-------------|----------|----------|
| ❄️ **冰封期** | 0-15 | 冰点期 | **0成** | 空仓，禁止操作 |
| 🥶 **过冷** | 15-30 | 退潮期末端 | **1-2成** | 仅做盘后复盘，不盘中操作 |
| 🌥️ **偏冷** | 30-45 | 启动期初期 | **2-3成** | 轻仓试探，只做领涨板块 |
| 🌤️ **中性** | 45-60 | 发酵期 | **4-5成** | 标准操作，顺势加仓 |
| ☀️ **偏热** | 60-80 | 高潮期初期 | **5-6成** | 主仓位已定，逐步减仓后排 |
| 🔥 **过热** | 80-90 | 高潮期末端 | **3-4成** | 只持有龙头，持续减仓 |
| 🚨 **极端过热** | 90-100 | 高潮期顶部 | **1-2成** | 只出不进，准备空仓 |

#### 温度计算公式

```
温度 = (
   涨停家数评分 * 25% +
   涨跌比评分 * 20% +
   主力资金评分 * 20% +
   V06期指基差评分 * 15% +
   V05板块净流入率评分 * 10% +
   V07 ETF资金流向评分 * 10%
)

各维度评分 0-100:
- 涨停家数: >100=100分, 80=80分, 50=50分, 30=30分, <15=0分
- 涨跌比: >2:1=100分, 1.5:1=70分, 1:1=50分, 1:2=20分, <1:3=0分
- 主力资金: >+300亿=100分, +100亿=60分, +0=40分, -200亿=20分, <-500亿=0分
- 期指基差: 全部升水=100分, 混合=60分, 全部贴水<0.3%=30分, 全部贴水>0.5%=0分
- 板块净流入率: max>3%=100分, >1%=60分, <0.3%=30分
- ETF资金: 净流入率>15%=100分, >5%=60分, <1%=20分
```

#### 仓位调整约束

| 调整场景 | 约束 |
|----------|------|
| 单日加仓上限 | 不超过2个交易单位（约2成仓位） |
| 连续加仓日上限 | 最多连续2天加仓 |
| 加仓后必须评估 | 加仓后次日必须检查温度是否支撑新仓位 |
| 温度下降1级 | 立即减仓至新温度等级的仓位上限 |
| 冰封期每日检查 | 9:30/11:30/14:00 三次检查冰封条件 |
