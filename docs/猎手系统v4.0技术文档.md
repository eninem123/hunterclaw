## 5. 恐慌指数系统

### 5.1 计算模型

恐慌指数(0-100) = 综合4因子评分(60为安全基准)

```
恐慌指数 = 60(基准) - 指数跌幅(30%) - 涨跌比(30%) - 量能(20%) - 连续缩量(20%)
```

| 恐慌指数 | 等级 | 交易限制 |
|---------|------|---------|
| ≥60 | 安全 | 正常交易 |
| 40-60 | 谨慎 | 轻仓控制 |
| 25-40 | 偏冷 | 仓位≤10% |
| 10-25 | 冰点 | 禁止新开仓 |
| <10 | 恐慌 | 强制空仓+全面禁止 |

### 5.2 应用场景

1. **R07-v2止损收紧**: 恐慌<40时止损收紧30%
2. **R08-v2止盈降档**: 恐慌<30时+2%强制减半
3. **R18冰点增强**: 恐慌<25联动V10强制空仓
4. **V09反弹检测**: 恐慌期反弹>2%辨识真假反弹
5. **信号降级**: 恐慌<40时买入信号置信度*0.8
