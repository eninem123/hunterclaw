## 2026-05-15: v3.15 规则升级

> **版本**: v3.15 | **状态**: ✅ 已部署 | **关联文档**: [优化总结_v2.md](./优化总结_v2.md)

### 概述

本次升级在原有v5.9策略框架基础上，引入 **R18-R21 四条核心规则**、**市场温度冻解体系**（Market Temperature Freeze/Thaw System）和 **V05-V07 三因子工具**，是策略从"经验驱动"向"温度量化驱动"的转折点。

### 新增规则 (R18-R21)

| 规则 | 名称 | 核心作用 | 数据依赖 |
|------|------|----------|----------|
| **R18** | 主力资金异常回流识别 | 区分真实反转vs诱多 | get_money_flow, V05 |
| **R19** | 板块内部强弱分化识别 | 判断板块健康度 | V05, get_hot_sectors |
| **R20** | 高位缩量加速识别 | 提前识别顶部风险 | get_daily_kline |
| **R21** | 市场冰封/解封系统 | 极端行情应急机制 | 所有V因子 + 市场情绪 |

### 市场温度冻解体系

#### 冰封条件（R21）

任一触发即进入**冰封期**：

| 条件编号 | 描述 | 数据源 |
|----------|------|--------|
| F1 | 沪深300单日跌幅 > 3% | get_index_quotes |
| F2 | 涨停 < 15家 且 跌停 > 50家 | get_market_overview |
| F3 | 主力资金单日净流出 > 500亿 | get_money_flow / V05 |
| F4 | 涨跌比 < 1:3 | get_market_overview |
| F5 | V06期指基差全部合约贴水 > 0.5% | get_futures_basis (V06) |

#### 解封条件（R21）

**全部**满足才解除冰封：

| 条件编号 | 描述 | 数据源 |
|----------|------|--------|
| T1 | 沪深300连续2日不再创新低 | get_index_quotes |
| T2 | 涨停回升至30家以上 | get_market_overview |
| T3 | 主力开始净流入 > 100亿 | V05 / get_money_flow |
| T4 | 涨跌比恢复至1:1以上 | get_market_overview |
| T5 | V05前3板块中有2个净流入率 > 1% | get_sector_money_flow_ratio (V05) |

#### 冰封期行为

| 维度 | 规则 |
|------|------|
| 新买入 | ❌ 禁止一切新买入 |
| 持仓操作 | 只执行止损，不执行止盈 |
| 仓位上限 | 强制降至2成以下 |
| 短线操作 | ❌ 禁止 |
| 最低持续 | 至少1个完整交易日 |
| 检查频率 | 9:30 / 11:30 / 14:00 三次 |

#### 冰封解除后恢复节奏

| 时间点 | 操作 |
|--------|------|
| 解封首日 | 仅恢复观察，不操作 |
| 解封次日 | 轻仓试探（1-2成） |
| 解封后第3日 | 确认企稳后恢复标准操作 |

### 新增因子工具 (V05-V07)

| 因子 | 工具名 | 全称 | 引入原因 |
|------|--------|------|----------|
| **V05** | `get_sector_money_flow_ratio` | 板块主力净流入率 | 替代北向实时信号（2026-05-13变更） |
| **V06** | `get_futures_basis` | 期指基差信号 | 市场情绪补充指标 |
| **V07** | `get_etf_money_flow` | ETF资金流向 | 机构资金配置信号 |

### 动态止损表（更新）

| 温度等级 | 硬止损 | 移动止损 |
|----------|--------|---------|
| ❄️ 冰封期 | -2% | 不适用 |

### 动态止盈表（更新）

| 温度等级 | 目标止盈 | 分批规则 |
|----------|---------|---------|
| ❄️ 冰封期 | 无止盈 | 只止损/减仓 |

| 🚨 极端过热 | +4% | 100%止盈 |

### 破坏性变更

- ❌ 无。v3.15完全向后兼容，但不建议在新版本之前使用V05-V07工具（旧版MCP未实现）。
- ⚠️ 市场温度系统与R2/R7/R8原有规则有重叠，建议优先使用温度表统一管理。

## 2026-05-13: 北向数据取消 & V04-V07 扩展

> **版本**: v5.6 (策略) + MCP v1.1 (工具)

### 北向资金重大调整

#### 变更背景
2026年5月13日，沪深交易所宣布北向资金（沪深港通/陆股通）**不再实时披露**，改为盘后（T+1）统计数据。这是A股历史上北向数据透明度的重要转折。

#### 影响范围

| 受影响系统 | 影响程度 | 说明 |
|-----------|----------|------|
| MCP工具 - get_north_money | 🔴 功能受限 | 仅T+1盘后可用 |
| MCP工具 - get_market_sentiment | 🟡 降权 | 北向部分使用前日数据 |
| MCP工具 - get_market_sentiment_enhanced | 🟡 降权 | 同上 |
| 策略规则R66/R90/R44/R40/R37 | 🔴 需修订 | 详见NB-001~NB-003 |
| 策略规则R46/R67/R92 | 🟡 降权 | 改为盘后参考 |

#### 替代方案

```
实时替代: 主力资金净流入/净流出（AKShare stock_individual_fund_flow）
盘后参考: 北向资金T+1数据（次日9:30前获取）
频率调整: 北向从盘中决策信号降级为日收盘检验
```

#### 规则修订清单

| 原规则 | 修订方式 | 新内容 |
|--------|---------|--------|
| R66 - 北向观望信号 | 删除实时依赖 | 改为盘后审查，作为次日仓位参考 |
| R90 - 北向流入<30亿触发 | 替代 | 改为"主力净流出>200亿"触发 |
| R44 - 北向转向低位板块 | 删除 | 保留"主力资金转向" |
| R40 - 北向席位验证 | 降权 | 改为"盘后席位分析" |
| R37 - 机构+北向共振 | 替代 | 改为"机构+主力共振买入" |
| R46 - 北向+机构+游资共振 | 替代 | 改为"主力+机构+游资共振" |
| R67 - 龙虎榜北向席位 | 降权 | 盘后参考，非盘中决策 |
| FM-003 - 北向重仓股预警 | 删除北向部分 | 保留板块内主力资金监控 |
| R34/R35 - 北向持股加分 | 删除 | 保留"机构持股"加分 |

### V05-V07 因子工具引入

#### 引入原因
北向数据取消后，需要新的实时数据源补充：

| 因子 | 对应MCP工具 | 上线版本 | 数据源 |
|------|------------|---------|--------|
| V05 | get_sector_money_flow_ratio | v1.1 | AKShare 板块资金流向 |
| V06 | get_futures_basis | v1.1 | AKShare 股指期货基差 |
| V07 | get_etf_money_flow | v1.1 | AKShare ETF资金流向 |

#### 更多同日变更
- 波动率交易量化（VOL-001~VOL-010）→ v5.3
- 策略审计修复：回滚R1.1/R87/R88/R89（假数据验证的规则）→ v5.4
- 规则回滚（命中率不足65%）：R19(57.1%)/R20(58.3%)/FM-004(60.0%)/FM-007(62.5%) → v5.5
- 北向规则修订完成 → v5.6
- AI虹吸效应FM-009 / 创业板突破格局FM-010 / 主力强度量化FM-011 → v5.8
- 假护盘识别SR-001 / 支撑衰减预警SR-002 → v5.9

### 破坏性变更

- 🔴 **北向资金相关规则全面修订**：涉及12条规则，其中4条删除、3条替代、2条降权、3条保留为盘后参考。
- ⚠️ 旧版策略文件（v5.5之前）中的北向规则不再有效。

## 2026-04-29: Level2 进阶规则（V01-V04）

> **版本**: v3.97 (策略) + MCP v1.0 (工具)

### 概述
Level2进阶规则是猎手系统从"基础选股"到"量化因子选股"的第一次跃升。引入**四个量化因子工具（V01-V04）**和**筹码集中度规则（CD-001~CD-010）**。

### 新增因子工具 (V01-V04)

| 因子 | 工具名 | 全称 | 来源 | 用途 |
|------|--------|------|------|------|
| **V01** | `get_money_flow` | 个股主力资金流向 | AKShare | 单只股票的主力净流入/出 |
| **V02** | `get_sector_money_flow` | 板块资金流向排名 | AKShare | 板块资金排名 |
| **V03** | `get_market_sentiment` | 综合市场情绪 | AKShare + 自定义 | 市场整体情绪评分 |
| **V04** | `get_market_sentiment_enhanced` | 增强情绪分析 | 多源聚合 | 深度情绪分析 |

> 注：V01-V04最初作为基础因子被引入，V05-V07是在2026-05-13北向数据变更后作为补充加入的。

### 新增规则 (CD-001~CD-010)

| 规则 | 名称 | 核心内容 |
|------|------|----------|
| CD-001 | 低位黄金区信号 | 筹码低位密集区域买入信号 |
| CD-002 | 放量突破信号 | 筹码单峰密集+放量突破买入 |
| CD-003 | 回踩支撑信号 | 筹码峰回踩确认支撑 |
| CD-004 | 高位分散卖出信号 | 高位筹码分散卖出 |
| CD-005 | 假集中真出货识别 | 识别虚假筹码集中 |
| CD-006 | 下峰消失止损信号 | 支撑筹码峰消失止损 |
| CD-007 | 趋势延续持有信号 | 趋势中持有条件 |
| CD-008 | 筹码仓位调节规则 | 筹码集中度仓位加减 |
| CD-009 | 筹码-猎手信号融合矩阵 | 筹码与其他信号评分矩阵 |
| CD-010 | 稀土ETF筹码监控方案 | 稀土ETF专项筹码监控 |

### 新增RSRS指标规则 (RS-001~RS-008)

| 规则 | 名称 | 核心内容 |
|------|------|----------|
| RS-001 | RSRS强烈买入信号 | 斜率>1+标准R²>0.7+放量 |
| RS-002 | RSRS立即卖出信号 | 斜率<-1+标准R²>0.7 |
| RS-003 | 收口突破买入规则 | 斜率走平+布林带收口+放量突破 |
| RS-004 | 开口顺势持有规则 | 斜率持续+布林带开口 |
| RS-005 | 量能确认原则 | 突破必须有量能支撑 |
| RS-006 | 中性观望信号 | 斜率|RSRS|<0.5观望 |
| RS-007 | 布林带状态判断 | 布林带收口/开口判断 |
| RS-008 | 综合共振评分 | RSRS+MACD+KDJ共振评分 |

### 新增网格交易规则 (WG-001~WG-011)

| 规则 | 名称 | 核心内容 |
|------|------|----------|
| WG-001 | 网格交易适用标的筛选 | 波动率>3%+日均成交>1亿+流通市值>50亿 |
| WG-002 | 网格区间设计标准 | 下行预留10%+上行预留10% |
| WG-003 | 网格资金分配规则 | 首笔净买30%+预留网格保证金30% |
| WG-004 | 网格买入执行规则 | 每跌3%买1笔，金额递增10% |
| WG-005 | 网格卖出执行规则 | 每涨3%卖1笔，先买先卖 |
| WG-006 | 网格止损规则 | 突破下轨-3%全仓止损 |
| WG-007 | 网格止盈规则 | 突破上轨启动偏移止盈 |
| WG-008 | 网格与猎手信号整合 | 网格底仓+猎手信号增强 |
| WG-009 | 网格动态优化规则 | 波动率变化时调整网格间距 |
| WG-010 | 网格交易铁律 | 不加杠杆、不扛单、不扩大网格 |
| WG-011 | 稀土ETF网格专项方案 | 稀土ETF专用网格参数 |
