### 2.2 data_analyzer_v2.py — 技术分析引擎 (v3.1)

**版本**: v3.1 (2026-05-17升级)  
**核心类**: `DataAnalyzerV2`

**v3.1新增指标**:
- **Stochastic RSI (StochRSI)**: 将RSI值二次标准化到0-1范围，比普通RSI更灵敏，精准识别超买超卖区域。阈值为 <0.15(深度超卖), <0.25(超卖), >0.75(超买), >0.85(深度超买)
- **ADX/DMI (平均趋向指数)**: 衡量趋势强度，ADX>25表示趋势行情，配合+DI/-DI判断多空方向。ADX<20表示震荡行情，ADX>40表示极强趋势（注意末端反转）
- **信号评分集成**: StochRSI和ADX纳入综合评分、KNN预测、市场机制检测

**支持技术指标 (完整清单)**:
| 类别 | 指标 | 说明 |
|------|------|------|
| 移动平均 | MA5/10/20/60/120/250 | 多周期均线 |
| 趋势 | RSI(14), MACD(12/26/9), DMA(10/50), TRIX(12) | 趋势方向与动能 |
| 趋势强度 | **ADX(14)**, **+DI/-DI** 🆕 | 趋势强度+多空方向 |
| 超买超卖 | KDJ(9/3/3), CCI(14), WR(14), **StochRSI(14/3/3)** 🆕 | 精准超买超卖识别 |
| 波动率 | BOLL(20/2), ATR(14) | 价格波动区间 |
| 量价 | OBV, 量比 | 量价配合 |
| 情绪 | PSY(12) | 市场心理 |

## 5. 恐慌指数系统

### 5.1 计算模型 (v4.0)

恐慌指数(0-100) = `60(基准) - 指数跌幅(30%) - 涨跌比(30%) - 量能(20%) - 连续缩量(20%)`

```
级别      范围    含义
──────────────────────
安全      ≥60    正常交易
谨慎     40-60   轻仓控制
偏冷     25-40   仓位≤10%
冰点     10-25   禁止新开仓
恐慌      <10    强制空仓+全面禁止
```

## 7. 新增指标说明 (v3.1)

### 7.1 Stochastic RSI

**公式**: `StochRSI = (当前RSI - 最低RSI_n) / (最高RSI_n - 最低RSI_n)`

用途: 对RSI做二次标准化，产生0-1的值。比RSI更敏感。
- < 0.15: 深度超卖（强烈买入信号）
- 0.15-0.25: 超卖
- 0.40-0.60: 中性
- 0.75-0.85: 超买
- > 0.85: 深度超买（强烈卖出信号）

### 7.2 ADX (Average Directional Index) / DMI

**公式**: ADX = SMA(|+DI - -DI| / (+DI + -DI)) × 100

用途: 判断趋势强度和方向。
- ADX < 20: 弱趋势/震荡行情
- ADX 20-25: 趋势形成中
- ADX 25-40: 强趋势行情（配合+DI/-DI判断方向）
- ADX > 40: 极强趋势（警惕末端加速后反转）

+DI > -DI: 多头主导；-DI > +DI: 空头主导
