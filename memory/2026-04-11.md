## 标准开发部署流程（必须遵循）

每次改代码后执行顺序：

1. **改源码** → `/root/Alpha_Terminal/`
2. **同步生产** → `rsync -av --exclude='.git' --exclude='node_modules' --exclude='.env*' /root/Alpha_Terminal/ /var/www/html/`
3. **重启服务** → `pkill -f "node server.js"; sleep 1; cd /var/www/html && nohup node server.js > /tmp/server.log 2>&1 &`
4. **推送 Git** → `cd /root/Alpha_Terminal && git add . && git commit -m "描述" && git push`

生产环境：服务器 `/var/www/html/`（IP: 已在 nginx 配置）
源码目录：`/root/Alpha_Terminal/`（GitHub 维护）
