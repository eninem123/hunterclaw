## 三、止损止盈策略调整（v3.10）

### 3.1 v3.9固定策略问题

```
止损：-5%固定  ← 不管市场温度，不管持仓时间
止盈：+8%固定  ← 不管波动率，不管持仓周期
```

**问题本质：** 忽略了市场环境差异和持仓时间价值

### 3.2 v3.10 动态止损止盈矩阵

#### 止损策略（按市场温度 × 持仓时间）

| 市场温度 | T+0~T+2 | T+3~T+5 | T+6~T+10 |
|---------|---------|---------|---------|
| ≥70℃ | -6% | -4%移动止损 | -3%移动止损 |
| 50-70℃ | -5% | -4%移动止损 | -3%移动止损 |
| 35-50℃ | -4% | -3%移动止损 | -2%移动止损 |
| 25-35℃ | -3% | -2%移动止损 | 成本价 |
| <25℃ | 空仓 | — | — |

#### 止盈策略（分批止盈）

| 市场温度 | 第一批 | 第二批 | 第三批 |
|---------|-------|-------|-------|
| ≥70℃ | +5%（1/3） | +10%（1/3） | +15%（1/3） |
| 50-70℃ | +4%（1/3） | +8%（1/3） | +12%（1/3） |
| 35-50℃ | +3%（1/3） | +6%（1/3） | +10%（1/3） |
| 25-35℃ | +2%（1/2） | +5%（1/2） | — |

#### 移动止损规则（关键创新）

```
触发条件：持仓浮盈≥+3%时，激活移动止损
执行逻辑：
  T+3起：将止损线上移至 min(成本价, 前日收盘价-2%)
  T+6起：将止损线上移至 min(成本价, 前高-4%)
  效果：让利润奔跑，但锁住大部分浮盈
```

```
# 规则名称：选股综合评分规则

# 优先级：P1

- 量比评分：≥1.3=1分, ≥2.0=2分, ≥3.0=3分

综合评分<60分 → 降级为观察
```

```
# 规则名称：隔夜持仓风险扫描
# 生效时间：每个交易日 22:00 自动执行
# 优先级：P1

[扫描逻辑]
1. 遍历所有持仓标的
2. 检查是否有跌停/停牌公告
3. 检查主力资金是否转为净流出
4. 计算止损/止盈线距离现价百分比
5. 判断市场温度预判（明日开盘）

[输出动作]
- 止损风险≥80%触发 → 明日09:15前发送预警，建议开盘执行
- 止盈条件已触发 → 建议开盘执行分批止盈

```
