## 四、猎手规则库更新（v3.12）

### 4.1 新增规则：高位长上影线过滤

```
# 规则名称：高位长上影线过滤规则
# 生效日期：2026-05-13 v3.12
# 优先级：P1

[规则描述]
当日上影线>3%的股票存在主力出货风险，应谨慎买入

[计算方法]
上影线比例 = (当日最高价 - 收盘价) / 收盘价 × 100%

[触发条件]
上影线比例 > 3% → 降级为观察，降低买入优先级
上影线比例 > 5% → 禁止买入

[例外情况]
- 涨停股的上影线不适用此规则（涨停本身已排除）
- 开盘跳空高开回落的影线需额外判断
```

### 4.2 新增规则：尾盘强势保留

```
# 规则名称：尾盘强势股保留规则
# 生效日期：2026-05-13 v3.12
# 优先级：P2

[规则描述]
尾盘强势股可考虑保留至次日开盘，获取隔夜溢价

[触发条件]
- 时间：14:30之后
- 涨幅：>3%
- 量比：>1.5
- 持仓状态：已触发第一批止盈但未执行

[执行逻辑]
if 尾盘强势 and 已触发第一批止盈:
    暂停执行止盈
    设置次日开盘必卖条件：
      - 开盘价 < 持仓成本 → 全部止损
      - 开盘涨幅 > 5% → 分批止盈
      - 其他情况 → 按市场温度执行常规止损止盈

[风险控制]
- 最多保留一次（连续保留禁止）
- 保留期间止损自动上移至成本价
```

### 4.3 新增规则：跳空低开保护

```
# 规则名称：跳空低开保护规则
# 生效日期：2026-05-13 v3.12
# 优先级：P1

[规则描述]
隔夜跳空低开>3%时，开盘即执行止损，避免进一步亏损

[触发条件]
开盘价 < 前日收盘价 × 0.97（即低开>3%）

[执行逻辑]
if 跳空低开 > 3%:
    开盘即执行止损（不计成本卖出）
    记录为"跳空止损"
    不计入当日买入次数限制

[例外情况]
- 若开盘后迅速反弹（5分钟内收复跌幅>2%）→ 可观望
- 板块整体低开（大盘低开）→ 按市场温度判断是否执行
```
