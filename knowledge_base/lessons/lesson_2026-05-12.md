## 四、猎手规则库更新（v3.11）

### 4.1 新增规则：追高限制

```
# 规则名称：买入价格偏离限制
# 生效日期：2026-05-12 v3.11
# 优先级：P1

[规则描述]
买入价不得超过信号触发时价格的+5%
超过则放弃买入或等回调

[例外情况]
- 开盘竞价时信号价格无法获取 → 按前日收盘价计算
- 强势股跳空高开 → 可放宽至+7%，但需二次确认

[验证逻辑]
if (买入价 - 信号价) / 信号价 > 5%:
    reject_or_wait()
```

### 4.2 新增规则：趋势持续性验证

```
# 规则名称：趋势持续性加分规则
# 生效日期：2026-05-12 v3.11
# 优先级：P1

[规则描述]
连续上涨趋势比单日暴涨更可靠

[评分标准]
- 连续上涨1天：+0分（普通信号）
- 连续上涨2天：+2分（趋势确认）
- 连续上涨3天+量比>1.5：+4分（强趋势）

[排除条件]
- 连续上涨但量比<1.0（缩量上涨）→ 趋势不可靠，扣2分
- 涨停次日（高开低走风险）→ 禁止买入
```

### 4.3 强化规则：波动率调整止损

```
# 规则名称：波动率调整止损规则
# 生效日期：2026-05-12 v3.11

[规则描述]
高波动环境下，收窄止损幅度避免被震出

[计算方法]
近5日波动率 = (最高价-最低价)/最低价
- 波动率<3%：正常止损
- 波动率3-5%：止损收窄0.5%
- 波动率>5%：止损收窄1%

[示例]
稀土ETF：
  近5日最高: ~1.45, 最低: ~1.32
  波动率 = (1.45-1.32)/1.32 = 9.8% > 5%
  → 止损收窄1%，即从-4%变为-3%
  → 实际止损位：1.308*(1-0.03) = 1.269
```
